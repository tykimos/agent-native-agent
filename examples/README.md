# ANL Examples

**ANL** means **Agent-Native Lifestyle**. In this repository, ANL is shown as concrete examples made with **ANA**, not as a separate product or repository.

ANA is an agent-native agent for Agent-Native Lifestyle. These examples should show how a user's way of working, learning, creating, consuming, or running daily routines changes when an agent can watch context, converse, decide, act, and improve the app over time.

## Case Format

Each example should cover:

- **Lifestyle Shift**: what changed in the user's daily or work routine
- **ANA**: what agent-native agent was built
- **Watch**: what the dashboard shows at a glance
- **Converse**: what the user asks the agent to do
- **Agentic Loop**: what the agent can judge, execute, and improve over time
- **Result**: what became easier, faster, or newly possible

## Example Domains

- Work queue and inbox operations
- Family or personal planning
- Learning and review loops
- Creative publishing workflows
- Order, customer, or approval desks
- Health, finance, travel, and subscription routines
