# ANL 예시

**한국어** · [English](README.md)

**ANL**은 **Agent-Native Lifestyle**을 뜻합니다. 이 저장소에서 ANL은 별도 제품이나 별도 리포지토리가 아니라, **ANA**로 만든 구체적인 사례로 보여줍니다.

ANA는 Agent-Native Lifestyle을 위한 agent-native agent입니다. 각 예시는 에이전트가 맥락을 보고, 대화하고, 판단하고, 실행하고, 시간이 지나며 앱을 개선할 때 사용자의 일상과 업무 방식이 어떻게 바뀌는지를 설명해야 합니다.

## 사례 형식

각 사례는 다음 내용을 다룹니다.

- **Lifestyle Shift**: 사용자의 일상 또는 업무 루틴이 어떻게 바뀌었는가
- **ANA**: 어떤 agent-native agent를 만들었는가
- **Watch**: 대시보드에서 무엇을 한눈에 보는가
- **Converse**: 사용자는 에이전트에게 무엇을 요청하는가
- **Agentic Loop**: 에이전트가 무엇을 판단하고, 실행하고, 반복 개선하는가
- **Result**: 무엇이 쉬워졌고, 빨라졌고, 새롭게 가능해졌는가

## 예시 도메인

- 업무 큐와 인박스 운영
- 가족 또는 개인 일정 운영
- 학습 계획과 반복 복습
- 창작과 퍼블리싱 워크플로우
- 주문, 고객, 결재 데스크
- 건강, 금융, 여행, 구독 루틴
